# AI-Assisted Insight Summaries

This folder documents the AI-assisted insight summary component of the project.

The project uses AI assistance only for insight communication. Python, SQL, and Power BI were used to clean, analyze, validate, and visualize the data first. After validation, AI assistance was used to draft concise stakeholder-friendly summaries from the verified dashboard metrics.

The final insight summaries are included in:

- `README.md`
- Power BI `Key Insights` page

AI was not used for diagnosis, prediction, automated clinical decision-making, or replacement of analytical validation.
