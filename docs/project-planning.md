# Project Planning

## Business Questions

This project focuses on hospital operations, clinical trends, financial performance, and patient analytics.

Key business questions:

- How is patient admission volume changing over time?
- Which departments have the highest workload?
- Which diseases and disease categories are most common?
- What is the average length of stay?
- What are the major revenue trends?
- How much revenue is covered by insurance vs paid by patients?
- Are there operational bottlenecks related to departments, wards, beds, or staff?

## Stakeholders

Primary stakeholders:

- Hospital leadership
- Operations managers
- Department heads
- Clinical quality teams
- Billing and finance teams
- Healthcare data analysts

## KPIs

Planned KPIs:

- total admissions
- total unique patients
- total revenue
- average length of stay
- admissions by department
- admissions by disease category
- emergency vs elective admissions
- insurance-covered amount
- patient payable amount
- payment status distribution
- bed status distribution
- doctor and staff workload indicators

## Dashboard Structure

The Power BI dashboard will have four planned pages:

1. Executive Dashboard
2. Clinical Dashboard
3. Operations Dashboard
4. Financial Dashboard

## Project Scope

Included:

- dataset understanding
- data quality review
- data cleaning and preprocessing
- SQL-based KPI analysis
- Python exploratory data analysis
- Power BI dashboard development
- AI-assisted insight summaries
- GitHub documentation

Not included initially:

- advanced machine learning
- real patient-level clinical decision support
- genomics, waveform, or ICU signal data
- production-level hospital system deployment

## Notes

The project is designed for healthcare analytics, clinical data analyst, and healthcare BI roles, especially with relevance to hospital analytics responsibilities in India.
